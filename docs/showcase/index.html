Comparison

为什么不直接拼 OKX Skills

直接拼 OKX Skills

  • token：找币
  • market：看行情
  • quote：拿报价
  • gateway：查订单
结果 需要自己编排，自己补风控

购买 Treasury Guard

  • policy：该不该做
  • trade：怎么入场
  • exit：怎么退场
  • watch：后续怎么盯
结果 直接拿决策结果，直接按次购买
